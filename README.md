# Universal Converter

**Universal Converter** – это графическое приложение для точного перевода неотрицательных рациональных чисел из одной позиционной системы счисления в другую.  
Программа поддерживает:

- целые числа и конечные дроби;
- периодические дроби (чистые и смешанные);
- цифры в системах счисления до 500;
- запись цифр с помощью букв A–Z (значения 10–35) и квадратных скобок `[число]` (значения ≥ 36);
- проверку корректности входных данных;
- длинную арифметику (собственные классы `BigInteger` и `BigFraction`);
- ограничение на длину выходной строки (`MAX_OUTPUT_LENGTH = 100000`);
- загрузку входных данных из файла и сохранение результата в файл.

---

## Требования

- **C++17** или новее
- **Qt** (версия 5 или 6)
- **CMake** (≥ 3.16)

### Зависимости

- QtCore, QtWidgets
- Стандартная библиотека C++
